# 🍪 おしゃべりクッキーのSUDOKU (Talking Cookie Sudoku)

**「修行の時間だッキー！最高の数独プレイヤーを目指すッキー！」**

おしゃべりなクッキーがナビゲートする、ユニークでスタイリッシュなWebベースの数独（ナンプレ）ゲームです。
ただのパズルではありません。**「爆速連打」**や**「マウスホイール追跡」**など、独自の高速入力機能を搭載し、これまでにない爽快なプレイ感を提供します。

![Game Screenshot](static/img/cookie_rich.png)

## ✨ 主な特徴 (Key Features)

### ⚡ 爆速連打 (Lightning Mode)
数字を先に選び、盤面をタップ（クリック）するだけで次々と数字を埋められるモードです。
「ここは1...ここも1...あそこも1！」といった具合に、直感的な連続入力が可能です。

### 🖱️ マウスホイール数字スキャン (Wheel & Fast Pencil)
「自動メモ (Fast Pencil)」モード中、マウスホイールを回すだけで、選択中の数字（および盤面上の同じ数字）を次々とハイライト切り替えできます。
「3はどこに入る？」を一瞬で視覚的にスキャンできる、最強の探索ツールです。

### 🗣️ おしゃべりクッキーのガイド
謎のキャラクター「おしゃべりクッキー」が、チュートリアルからヒント、ゲームオーバー時の励ましの言葉まで、全編にわたってプレイヤーをサポート（？）します。
3回ミスすると問答無用でゲームオーバーになるので注意！

### 🎨 プレミアムなデザイン
- グラスモフィズム（すりガラス風）を取り入れたモダンなUI
- プレイを盛り上げるアニメーションと紙吹雪演出
- ダークモード / ライトモード対応

## 🚀 セットアップ (Installation)

### 必要要件
- Python 3.8+
- Flask

### インストール手順

1. リポジトリをクローンします:
   ```bash
   git clone https://github.com/konbu-hub/sudoku.git
   cd sudoku
   ```

2. 依存パッケージをインストールします:
   ```bash
   pip install -r requirements.txt
   ```
   *(基本的に `Flask` があれば動作します)*

3. アプリケーションを起動します:
   ```bash
   python app.py
   ```

4. ブラウザでアクセスします:
   `http://127.0.0.1:5000`

## 🎮 遊びかた

1. **難易度選択**: 「やわらかクッキー（Easy）」から「石（Extreme）」まで選べます。
2. **基本操作**:
   - マウス: クリックで選択、パネルで入力。
   - キーボード: 矢印キーで移動、数字キーで入力、BSで消去。
   - マウスホイール: 数字の切り替え（Fast Pencil時）。
3. **ヒント**: どうしてもわからない時は「ヒント」を使えますが、**1回のプレイにつき3回まで**です。
4. **ゲームオーバー**: **3回ミス**すると即終了です。慎重に！
