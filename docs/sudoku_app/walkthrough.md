# おしゃべりクッキーの数独 アップデート概要

「おしゃべりクッキーの数独」を全面的にアップグレードし、より洗練されたゲームアプリへと進化させました。最終的なブランド名は「クボゲー Vol.1」となりました。

## 主な変更点

### 1. メインメニュー（タイトル画面）の導入
アプリ起動時に「クボゲー Vol.1」というタイトルと、闇落ちしたおしゃべりクッキーが迎えてくれるタイトル画面を表示します。
ここからゲーム開始、ランキング確認、名前変更、オプション設定が行えます。

### 2. システム設定とテーマ切り替え
- **テーマ設定**: ダークモードとライトモードが選択可能になり、設定は保存されます。
- **初期モード設定**: ライトニングモードやファストペンシルを最初からONにした状態でゲームを開始できるようになりました。

### 3. ランキングの強化
- 難易度ごとにタブを切り替えてランキングを確認できる機能を追加しました。

### 4. 演出と利便性の向上
- **クリア演出**: クリア時に画面いっぱいに紙吹雪が舞う演出を追加しました。
- **オートステップ**: ライトニングモード中、一つの数字が埋まった際に自動で次の数字へ切り替わる機能を追加しました。
- **UI調整**: タイマー・エラー表示のレイアウト固定などを行いました。

### 5. バグ修正
- ライトニングモードでの無限ループ（フリーズ）の解消。
- ファストペンシル使用時に候補数字が正しく表示されない問題の修正。
